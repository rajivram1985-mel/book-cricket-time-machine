Privacy Policy — Book Cricket Time Machine

Last updated: 10 July 2026

Book Cricket Time Machine is a free, nostalgic browser game. The short version: there are no accounts, no ads, no cookies, and your scorebook never leaves your device. The only thing we collect is an anonymous count of which parts of the game get used — never who used them.

What data the app collects

The app uses Umami, a privacy-focused analytics service, to count a small number of anonymous events — things like "a Classic match started," "the Daily Challenge was shared," or "the how-to-play panel was opened." Each event carries only a coarse category (which game mode, win or loss) and never anything that could identify you: no name, no email, no exact location, no device fingerprint. Umami doesn't use cookies and doesn't assign you a persistent ID that could track you across visits or days — there is no way for us, or anyone, to look at this data and see "what one specific person did." We only ever see aggregate counts, like "the Daily Challenge was started 40 times this week."

The app has no user accounts, no sign-in, and no advertising of any kind.

What's stored on your device

Your scorebook — matches played, win/loss record, daily-challenge streaks, personal bests and sound preferences — is saved in your browser's local storage, on your device only. It is never included in any analytics event and never transmitted anywhere. You can wipe it at any time with the “Reset data” link in the app's footer.

Backups

The “Back up” button saves your scorebook as a file to your own device's downloads. That file goes wherever you choose to put it — it is never uploaded anywhere by the app.

Network use

The app downloads its own game files (pages, images, commentary audio clips) from our web host when you first load or update it, like any website. These requests carry no personal information beyond what any standard web request contains. After the first visit, the game also works offline. Separately, the anonymous usage events described above are sent to Umami's servers; nothing else the app does involves a network request to anyone besides our own host and Umami.

Children

The game is safe for all ages. The anonymous usage counts described above never include anything that could identify an individual player, child or adult.

Changes to this policy

If the app's data practices ever change (for example, if optional online features are added someday), this page will be updated before those changes ship, and the app's store listing will reflect it.
